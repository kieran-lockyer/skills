# MBA Study Vault

## How this vault is organised

```
Vault/
├── README.md                            — You are here
├── Templates/                           — Reusable templates for new notes (Home, Module, Concept, Theorist, Example)
└── BUSN20027 — Contemporary Leadership Vault/
    ├── Home.md                          — Main map of content (MOC)
    ├── 00 Unit/                         — Unit overview, assessment brief, rubric
    ├── 01 Modules/                      — One note per module (1–7)
    ├── 02 Concepts/                     — Leadership theories and frameworks
    ├── 03 Theorists/                    — Notes on thinkers (Burns, Bass, Dweck, Deming, etc.)
    ├── 04 Examples/                     — Australian & NFP real-world examples
    └── 05 Readings/                     — (Reserved for summaries of individual peer-reviewed PDFs)
```

## How to use it

1. Start at **[[Home]]** for the overall map.
2. Work through a module note (e.g., [[Module 1 - Why Leadership Matters]]). Each module note follows this structure:
   - **One-line summary** — the module's central thesis
   - **Sub-topics** — the numbered sections (1.1, 1.2, …) with embedded links to the concepts they introduce
   - **Key concepts introduced** — links to concept notes
   - **Key theorists introduced** — links to theorist notes
   - **Australian / NFP examples** — links to example notes, each tagged with the leadership angle it illustrates
   - **Strategic takeaway for the presentation** — how the module maps to a rubric criterion
   - **Source readings** — peer-reviewed articles from the module folder
3. Concept notes follow the pattern: *plain-language definition → why it matters for strategic decision-making → real-world example*.
4. Use **backlinks** on any concept or theorist to see every place it appears.
5. Graph view: shows the "theory cluster" map — useful when choosing a media article for the Presentation.
